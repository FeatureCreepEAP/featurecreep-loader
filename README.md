# featurecreep-loader

JBoss Modules based modloader.
Requires JBoss Modules
Works on Java 8-current
USE JAVA 11+ FOR BUILDING WITH MVN COMMAND
Has transformer support
Very easy to implement 

Only FCLoaderBasicR8 works others are obsolete
